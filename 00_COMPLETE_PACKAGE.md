## ğŸ†˜ TROUBLESHOOTING

### If DEPLOY.bat fails:
- Check AWS credentials: `aws sts get-caller-identity`
- Check Terraform installed: `terraform version`
- Follow [DEPLOYMENT_STEPS.md](DEPLOYMENT_STEPS.md) manually

### If GitHub Actions fails:
- Check AWS_ROLE_ARN secret in GitHub
- Verify secret value matches exactly
- Check OIDC provider exists

### If website shows 403:
- Run `terraform apply` again
- Wait 1-2 minutes for S3 permissions to sync
- Check bucket policy

### If files didn't upload:
- Check bucket exists: `aws s3 ls`
- Manually upload: `aws s3 cp index.html s3://noor-portfolio-website/`
