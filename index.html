Portfolio Website

Deployed with Terraform & GitHub Actions

Welcome!

This is a static website deployed to AWS S3 using Terraform Infrastructure as Code and GitHub Actions CI/CD pipeline.

Technologies Used

  • Terraform - Infrastructure as Code
  • AWS S3 - Static Website Hosting
  • GitHub Actions - CI/CD Pipeline
  • HTML/CSS - Frontend

Key Features

✓ Infrastructure as Code ✓ Automated Deployment ✓ Responsive Design ✓ Fast & Scalable

How It Works

Push code to GitHub → GitHub Actions validates Terraform → Deploys to S3 → Website goes live automatically!
